# Data Directory

## METR-LA

| File                         | Description                                                                           |
| ---------------------------- | ------------------------------------------------------------------------------------- |
| `metr-la.h5`                 | Traffic speed readings — 207 sensors × 34,272 time steps (5-min intervals, ~4 months) |
| `adj_mx.pkl`                 | Distance-based adjacency matrix (sensor_ids, index_map, 207×207 weight matrix)        |
| `graph_sensor_locations.csv` | Sensor GPS coordinates (latitude, longitude)                                          |
| `adj_mx.npy`                 | Dense numpy adjacency (generated by `build_graph.py`)                                 |
| `node_coords.csv`            | Formatted sensor coordinates                                                          |
| `graph.gpickle`              | NetworkX DiGraph                                                                      |
| `graph_map.html`             | Interactive Folium city map                                                           |

## PeMS-BAY

| File             | Description                                         |
| ---------------- | --------------------------------------------------- |
| `pems-bay.h5`    | Traffic speed — 325 sensors, San Francisco Bay Area |
| `adj_mx_bay.pkl` | Adjacency matrix                                    |

## Download

```bash
python data/download_data.py --dataset metr-la
python data/download_data.py --dataset pems-bay
# or both:
python data/download_data.py --dataset all
```

## Build graph artefacts

```bash
python src/build_graph.py --city metr-la
python src/build_graph.py --city algiers --nodes 150
```
